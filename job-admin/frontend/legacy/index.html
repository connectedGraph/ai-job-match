System Constraints / 当前系统口径

这几个规则已经固定下来，后续实现直接围绕它展开。

1. 岗位库支持增删改查 2. 上传构建是后台的一个功能页，不是独立系统 3. 上传后要自动画像化 4. Tag 归一只对历史高频 canonical tag 做候选匹配 5. 如果是第一次上传，没有历史 tag，则不归一 6. 归一只回填 `normalizedTag` 与 `freq`，不再写回旧版归一化痕迹字段
